# PLC-STM-ESP Control

This project focuses on industrial control tests using PLC, STM32 Nucleo and ESP platforms.

## Project Scope
The project includes:
- PLC based control structure
- STM32 Nucleo board integration
- ESP-based support modules
- Ethernet communication tests
- Industrial control system development

## First Stage
In the first stage, Ethernet communication tests are being carried out on the STM32 Nucleo platform.

## Goal
The goal is to build a reliable control and communication infrastructure for industrial automation scenarios.

## Status
Project is under development.
